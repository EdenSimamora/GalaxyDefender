# Galaxy Defender

A simple space shooter game made with Python and Pygame.

## Features

* Move left and right
* Shoot enemies
* Score system
* Accuracy statistics
* Game Over screen
* Lightweight and beginner-friendly code

## Requirements

* Python 3.13+
* Pygame

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Controls

* Left Arrow = Move Left
* Right Arrow = Move Right
* Space = Shoot

## Statistics

After the game ends, the terminal displays:

* Final Score
* Enemies Killed
* Shots Fired
* Accuracy
